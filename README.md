# Info Buku Tugas

## Deskripsi
Project web sederhana berbasis PHP untuk menampilkan informasi beberapa buku dalam bentuk tampilan card menggunakan Bootstrap.

## Fitur
- Menampilkan daftar buku secara statis
- Informasi lengkap (judul, pengarang, penerbit, tahun, ISBN, harga, stok, dll)
- Format harga otomatis (Rupiah)
- Tampilan responsive dengan Bootstrap

## Teknologi
- PHP (Native)
- HTML
- Bootstrap 5

## Cara Menjalankan
1. Pastikan sudah menginstall XAMPP / Laragon
2. Simpan project di folder `htdocs`
3. Jalankan Apache
4. Akses melalui browser: http://localhost/nama-folder

## Tujuan
Project ini dibuat untuk memahami dasar penggunaan PHP dalam menampilkan data ke dalam HTML serta penggunaan Bootstrap untuk tampilan UI.

## Catatan
Data buku masih bersifat statis (hardcoded) dan belum menggunakan database atau struktur array dinamis.
